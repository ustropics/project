Tropical Cyclone Tracking Map

An interactive tropical cyclone tracking map. Ditch the pencil and paper, and insert your own custom tracks! This uses plotly as the main engine for drawing, and uses an X, Y, Z coordinate system for plotting (each plot can be interacted with on the map). X value = longitude, Y value = latitude, and Z value = wind speed ().

JS Fiddle Github Repository

Developer Notes: My major is meteorology, so I wanted to build something that would tell a story. There is no interactive plotter online that can easily create storm tracks, so thought this would be a fun project. I've done similar projects in Python and Jupyter notebook using model data (and matplotlibs/plotly), but just discovered plotly has a fully functional javascript library. This project utilizes that library with custom settings. The user can select a tropical cyclone to intialize the map or enter their own coordinates and hit Build Map.
